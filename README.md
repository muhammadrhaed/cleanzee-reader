# Cleanzee Reader

**The cleanest way to read Wikipedia. Zero ads, zero sidebars, zero distractions.**

Cleanzee Reader removes all clutter from Wikipedia articles so you can focus on what matters: the content.

### ✨ Features
- **Instant Clean View** - Paste any Wikipedia link, get clean text in 2 seconds
- **Zero Ads** - No popups, no banners, no distractions  
- **Mobile First** - Works perfectly on any phone
- **Formal Reading Mode** - Clean typography for focused studying
- **Add to Home Screen** - Install as PWA, works like a native app
- **100% Free** - No paywall, no sign-up, no tracking

### 🎯 Perfect For
- Students doing research
- Teachers preparing lessons  
- Anyone tired of cluttered websites
- Reading on small screens

### 🚀 How to Use
1. Go to https://muhammadrhaed.github.io/cleanzee-reader
2. Paste any Wikipedia link
3. Tap "Get Article" 
4. Read in peace

### 📱 Install as App
On Chrome mobile: Tap `⋮` → "Add to Home screen"
